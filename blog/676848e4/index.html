css常用选择器

发表于 2017-07-10 | 分类于 CSS ， 选择器 | 阅读需要 5 分钟

    CSS是 Cascading Style Sheet (层叠样式表)的缩写。 cascade （重叠）表示CSS的冲突规则，即如果有样式冲突，则将新的样式覆盖在旧的样式之上。

CSS规则结构

每个CSS规则都有两个基本部分，h1 表示选择器(selector) ，{color:deepskyblue;} 表示声明块(declaration block)。那么CSS中都有哪些常用的选择器呢？CSS中选择器基本分为五类：基本选择器、关系选择器、属性选择器、伪类选择器和伪元素选择器。

基本选择器

基本选择器主要由通配符选择器、元素（标签）选择器、ID选择器和类名选择器组成。

通配符选择器( * )

通配符选择器，你可以使用 * 命中所有的元素。出于对性能的考虑，通常不建议使用它。

元素（标签）选择器

元素选择器，是直接使用Html的标签元素作为选择器。例如: h1 {color:deepskyblue;} ,h1这个选择器会命中Html文档中所有的h1元素，将所有h1元素的字体颜色设置为red。

ID选择器

注意：Html元素的id属性是唯一的，这意味这你不能给不同的元素使用相同的id属性值。

使用ID选择器前,必须给Html的标签加上一个名为 id 的属性，该属性的值必须是合法的标识符,然后在样式表中通过 # + id属性值 来选中元素；

类名选择器

类名选择器允许你命中一批元素,但前提是你必须给他们加上一个名为 class的属性，并设置相同的属性值，同样该属性的值必须是合法的标识符,然后在样式表中通过 . + class属性值来选中元素。

关系选择器

关系选择器包括：后代选择器、子元素选择器、相邻兄弟选择器和普通兄弟选择器。

后代选择器

S 表示 选择器( Selector )

后代选择器的语法通常为：S1 S2 ,在两个选择器之间存在一个空格,该选择器命中所有被S1元素包含的S2元素;

子元素选择器

子元素选择器的语法通常为：S1 > S2 ,该选择器命中被S1元素包含的第一层S2元素;

相邻兄弟选择器

相邻兄弟选择器的语法通常为：S1 + S2 ,该选择器命中紧挨着S1元素，并且在S1之后的S2元素;

普通兄弟选择器

普通兄弟选择器的语法通常为：S1 ~ S2 ,该选择器命中所有在S1之后的S2元素;

属性选择器

属性选择器是通过元素中已经存在的属性名或属性值匹配元素。

[attr]

[attr] 将会选中页面中带有attr属性的元素，无论该元素是否有值。

[attr=”value”]

[attr="value"] 将会选中页面中带有attr属性，且属性值为value的元素。

<style>
    li[class="num-zero"] {
        background:deepskyblue;
    }
</style>
 ...
<li class="num-zero" >0</li>

[attr~=”value”]

[attr~="value"] 将会选中页面中带有attr属性，且属性值是一个以空格分隔的属性值列表，而其中至少有一个属性值为value。

<style>
    li[class~="num-zero"] {
        background:deepskyblue;
    }
</style>
 ...
<li class="num-zero zero" >0</li>

[attr^=”value”]

[attr^="value"] 将会选中页面中带有attr属性，且属性值是以value开头的元素；

<style>
    li[class^="num"] {
        background:deepskyblue;
    }
</style>
 ...
<li class="num-zero" >0</li>

[attr|=”value”]

[attr|="value"] 将会选中带有attr属性,且属性值是以value或者value- 开头的元素；

<style>
    li[class|="num"] {
        background:deepskyblue;
    }
</style>
 ...
<li class="num-zero" >0</li>

[attr$=”value”]

[attr$="value"] 将会选中带有attr属性,且属性值已value结尾的元素

<style>
    li[class$="zero"] {
        background:deepskyblue;
    }
</style>
 ...
<li class="num-zero" >0</li>

[attr*=”value”]

[attr*="value"] 将会选中带有attr属性，且属性值中包含value的元素

<style>
    li[class*="zero"] {
        background:deepskyblue;
    }
</style>
 ...
<li class="num-zero" >0</li>
